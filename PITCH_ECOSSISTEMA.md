# Pitch de Venda — Ecossistema Controle Total

---

## O Problema

O prestador de serviço (João) perde dinheiro todos os dias porque:

- Cliente quer um orçamento → João tá atendendo outro → perdeu o lead
- Cliente quer agendar → só por telefone → João não atendeu → perdeu
- Cliente quer remarcar → mesma história
- Cliente quer saber o preço → mesma história

**Resultado:** João perde horas no telefone e ainda perde cliente.

---

## A Solução — 3 Produtos, 1 Ecossistema

### 1. Vitrine — R$ 29/mês
O cliente do João encontra ele na internet.

```
meusite.controletotal.app/v/joao-encanador
┌──────────────────────────────────────┐
│  🪛 João Encanador · ★ 4.8          │
│                                      │
│  Serviços:                           │
│  ┌ Desentupir pia ────── R$ 120 ──┐ │
│  │ [Agendar ▶]  [📞 WhatsApp]     │ │
│  └─────────────────────────────────┘ │
│  ┌ Troca de chuveiro ── R$ 250 ───┐ │
│  │ [Agendar ▶]  [📞 WhatsApp]     │ │
│  └─────────────────────────────────┘ │
└──────────────────────────────────────┘
```

➡️ Cliente vê os serviços, preços e já pode **agendar sozinho**.

### 2. Atendente — R$ 47 ou R$ 97/mês
Se o cliente do João preferir WhatsApp, uma IA atende.

```
Maria: "João, o serviço inclui a peça?"
IA:   "Olá Maria! A troca do chuveiro inclui mão de obra.
      A peça você pode comprar ou eu posso levar.
      Confirmado para terça 10h! 👍"

Maria: "Posso mudar pra quarta?"
IA:   "Claro! Verifiquei aqui e quarta 10h está livre.
      Remarcado! ✅"
```

➡️ **João não precisa atender telefone nunca mais. A IA resolve.**

### 3. Controle Total — ERP (já existe)
A base que conecta tudo:

- Clientes, agendamentos, serviços no mesmo banco
- A Vitrine consulta a agenda em tempo real
- O Atendente usa os mesmos dados pra responder no WhatsApp
